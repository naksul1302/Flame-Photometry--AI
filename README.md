# AI-Powered Flame Photometer Dashboard

## Overview

This project is an AI-assisted Streamlit dashboard developed for the comparative analysis of sodium content in potato chip samples using Flame Photometry.

## Features

- NaCl Calibration Curve
- Automatic Regression Analysis
- Sodium Concentration Calculator
- Sample 1 Analysis
- Sample 2 Analysis
- Comparative Analysis
- Interactive Graphs
- AI-powered Scientific Interpretation (Gemini)
- Health Recommendation
- Automatic Lab Report Generation
- CSV and Excel Export

## Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
- NumPy
- SciPy
- Scikit-learn
- Google Gemini API
